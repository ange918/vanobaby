━━━ ÉTAPE 1 — API ROUTE (server-side token) ━━━━━━━━━━━━━
Create app/api/spotify/route.ts

Use Spotify Client Credentials Flow (no user login needed):
POST https://accounts.spotify.com/api/token
  body: grant_type=client_credentials
  headers: Authorization: Basic base64(CLIENT_ID:CLIENT_SECRET)

Then fetch these 3 endpoints with the token:

1. Artist info:
   GET https://api.spotify.com/v1/artists/6VxXJZxxq0cmpBvbVM8p0E
   → returns: name, followers.total, images[], popularity, genres[]

2. Top tracks:
   GET https://api.spotify.com/v1/artists/6VxXJZxxq0cmpBvbVM8p0E/top-tracks?market=BJ
   → returns array of tracks, each with:
      id, name, duration_ms, popularity,
      preview_url (30s audio clip — can be null),
      album.name, album.images[],
      external_urls.spotify (link to open in Spotify)

3. Albums:
   GET https://api.spotify.com/v1/artists/6VxXJZxxq0cmpBvbVM8p0E/albums?include_groups=single,album&market=BJ&limit=20
   → returns: id, name, release_date, images[], total_tracks, external_urls.spotify

Return all 3 as a single JSON response:
  { artist, topTracks, albums }

Add Next.js cache revalidation: revalidate every 3600 seconds (1 hour)
